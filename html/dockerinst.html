Instalación y Ejecución de Docker en Windows

Requisitos Previos

  • Windows 10/11 de 64 bits (Pro, Enterprise o Education). Si usas Home, necesitarás WSL 2 (Windows Subsystem for Linux).
  • Virtualización habilitada en la BIOS.

Proceso de Instalación

  1. Descarga el instalador de Docker Desktop desde la página oficial de Docker.
  2. Ejecuta el archivo Docker Desktop Installer.exe.
  3. Sigue las instrucciones del asistente. Asegúrate de marcar la opción para instalar WSL 2 si te lo solicita (es recomendado para mejor rendimiento).
  4. Una vez finalizada la instalación, reinicia tu computadora.
  5. Abre la aplicación "Docker Desktop". Acepta los términos y condiciones.

Ejecución y Comprobación

Para comprobar que Docker está funcionando correctamente, abre tu terminal (PowerShell o CMD) y ejecuta:

docker run hello-world

Este comando descargará una imagen de prueba y mostrará un mensaje de confirmación, indicando que el servicio está activo y listo para usarse.
